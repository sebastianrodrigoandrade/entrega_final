API Documentación

Productos

GET /api/products
Parámetros: limit, page, sort, query
POST /api/products
Body: Array de productos
PUT /api/products/:id
Body: Producto actualizado
DELETE /api/products/:id

Body: Array de productos
PUT /api/carts/:cid/products/:pid
Body: quantity
DELETE /api/carts/:cid
GET /api/carts/:cid
